Line

class Line

In the following constructors, additional keyword arguments can be provided to set properties of the new Line.

Line(p, q)

Construct the line passing through the two points p and q.

Line(m, c)

Construct the line with equation y = mx + c.

A Line has the following common properties:

  • is_visible

  • color

  • color_floats

  • line_thickness
