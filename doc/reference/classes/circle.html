Circle

class Circle

In the following constructors, additional keyword arguments can be provided to set properties of the new Circle.

Circle(x, y, r)

Construct the circle whose centre is (x, y) and whose radius is r.

Circle(p, r)

Construct the circle whose centre is the point p and whose radius is r.

Circle(p1, p2)

Construct the circle whose centre is the point p1 and which passes through the point p2.

Circle(p1, p2, p3)

Construct the circle which passes through the three points p1, p2, and p3.

property radius

(Get-only) The circle’s radius, as a Python float.

property radius_number

(Get-only) The circle’s radius, as a dynamic Number.

A Circle also has the following common properties:

  • is_visible

  • color

  • color_floats

  • line_thickness

  • opacity
