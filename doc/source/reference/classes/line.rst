Line
====

.. py:class:: Line

   In the following constructors, additional keyword arguments can be
   provided to set properties of the new :py:class:`Line`.

   .. py:method:: Line(p, q)

      Construct the line passing through the two points *p* and *q*.

   .. py:method:: Line(m, c)

      Construct the line with equation *y = mx + c*.

   A :py:class:`Line` has the following common properties:

   * :py:attr:`is_visible`
   * :py:attr:`color`
   * :py:attr:`color_floats`
   * :py:attr:`line_thickness`
